Integration

One script tag.
That's it.

Drop a single line of code on your website. We handle page views, form submissions, UTM capture, and session tracking automatically.

01

Add the script

One line in your HTML. The script loads asynchronously — zero impact on page speed.

<script src="https://your-domain.com /api/tracking/script?site=yoursite.com"></script>
02

Track form submissions

Call one function when your form submits. Name, email, phone — we encrypt and store everything.

// On form submit: PeopleDown.trackLead({ name: formData.name, email: formData.email, phone: formData.phone })
03

See it in real time

Leads appear in your dashboard instantly. Full attribution chain — from ad click to form fill — visible on every lead.

Security

Enterprise security.
No enterprise price.

PII encryption, audit trails, rate limiting, and role-based access — built in from day one, not bolted on.

🔐 Data Encryption

  • AES-256-GCM field-level encryption for PII
  • Unique IV per encryption operation
  • HMAC-SHA256 searchable email hashes
  • Encryption keys never leave the server

📋 Audit Logging

  • Every PII decryption logged with user + IP
  • Login attempts tracked (success + failure)
  • Export actions audit-trailed
  • Admin-only security dashboard

🛡️ Access Control

  • Role-based permissions (admin / user)
  • Copy protection for non-admin users
  • Export and sync restricted to admins
  • Session management with version control

⚡ Rate Limiting

  • API rate limiting: 100 req/min
  • Login brute-force protection: 5 attempts/5 min
  • Account lockout after failed attempts
  • HMAC webhook signature verification
